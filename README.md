# springboot-doma-todo

## Database
```
$ ./gradlew flywayMigrate
$ ./gradlew flywayInfo
```

## Build
```
$ ./gradlew build
```

## Usage
for develop
```
$ ./gradlew bootRun
```

for production
```
$ java -jar build/libs/springboot-doma-todo-0.0.1-SNAPSHOT.jar
```

## Develop with Idea
```
$ ./gradlew idea
```

## Develop with Eclipse
```
$ ./gradlew eclipse
```
